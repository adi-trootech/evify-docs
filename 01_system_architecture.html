Overview

The Evify Gamification & Reward Engine is a monolithic, event-driven SaaS platform built for delivery rider apps. It is designed to scale to 100,000+ riders across unlimited tenants with zero-code configurability for adding new parameters, rules, rewards, or campaigns. It utilizes a powerful visual workflow automation approach (via json-rules-engine) where Triggers, Conditions, and Actions are easily stitched together by operations teams without requiring code changes.

Multi-Tenant Isolation

Each tenant (delivery company) operates in an independently isolated namespace:

  • MongoDB: Separate database per tenant (tenant_acme, tenant_zomato)
  • Redis: Tenant-prefixed keys (t:{tenantId}:score:{riderId})
  • Background Jobs: Tenant-tagged queues or jobs in BullMQ
  • API Layer: JWT claims carry tenantId on every request
  • Admin Panel: Role-Based Access Control (RBAC) per tenant
